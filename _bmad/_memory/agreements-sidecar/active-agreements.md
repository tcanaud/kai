# Active Agreements

This sidecar file is maintained by the Agreement System.
BMAD agents can read this file to know which Agreements exist and their current status.

## How to use

When a user discusses a feature, check if its feature_id appears below.
If yes, reference the Agreement in your responses and remind the user
to keep it aligned after any product changes.
